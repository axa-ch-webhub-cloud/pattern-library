## Usage

Buttons communicate actions that users can take. They are typically placed throughout your UI, in places like:

- Dialogs
- Modal windows
- Forms
- Cards
- Toolbars

## Button

The `Button` component is the most common button component.
It contains a label and optional icons before or after the label.

```jsx
<Button>Hello</Button>
```

## All button variations

```jsx
<div>
  <div
    style={{
      display: 'flex',
      flexDirection: 'column',
    }}
  >
    <Button variant="secondary" marginRight={16}>
      secondary
    </Button>
    <Button variant="red" marginRight={16}>
      red
    </Button>
  </div>
  <div
    style={{
      backgroundColor: '#1e3952',
      display: 'flex',
      flexDirection: 'column',
    }}
  >
    <Button variant="inverted">inverted</Button>
    <Button variant="inverted-blue-ocean">inverted-blue-ocean</Button>
    <Button variant="inverted-red-tosca">inverted-red-tosca</Button>
    <Button variant="inverted-purple-logan">inverted-purple-logan</Button>
    <Button variant="inverted-green-viridian">inverted-green-viridian</Button>
    <Button variant="inverted-blue-teal">inverted-blue-teal</Button>
  </div>
</div>
```
